Custom HTML tags


I will start by mentioning that for most uses, custom HTML tags are considered to be bad practace, but for certain purposes, creating a custom tag can make writting repetitive code easier, and can make your code eiser for other developers to use. For this introduction, I decided to use the extreamly simple but never used (as far as I can find) example that could be applied to mush more robust projects.

First thing that you should know before beginning is that when you use a non existant tag in HTML, CSS, or the javacript "getElementsByTageName()", the browser will not throw an error, but instead assume that you are dfining your own tag. This empty tag will be defined as something very similar to that of a div; an empty box with very few properties associated with it. This allows you to simply create a tag with the normal HTML syntax, and then begin maunipulating it with CSS and JavaScript.

To demonstrate this proccess, we will be creating an easy way to add tabs to a page using the custom 't' tag.

<html>

     <head>

          <style>



          </style>

          <script>

               window.onload = function(){
                    var tags = document.getElementsByTagName("t");
                    for (var i = 0; i < tags.length; i++){
                         tags[i].innerHTML = "&nbsp;&nbsp&nbsp&nbsp;&nbsp;&nbsp;" + tags[i].innerHTML;
                    }
               }

          </script>


     </head>

     <body>
          <t>This line is now tabbed by five spaces.
     </body>

</html>

In this code, we simply manipulate the DOM as if the 't' tag was a true tag, and then write the HTML exactly as we would expect. In addition, we could define the tag further through CSS attributes.
