Custom Data Types with JavaScript


Javascript allows us to create new data types using the prototype and function objects. Custom data types exist in some form in nearly every major object oriented language(usually referred to as 'class') and can help a programmer to write quicker, more efficient code that is easier to both read and write.


There are three common ways to create objects in JavaScript, but for this tutorial, I will be teaching how to define a constructor function to form your objects. This is, in my opinion, easier to learn, read, and write due to its familiar syntax.


To create a new data type you simply define it as if it were a normal function, and then use the 'new' keyword to create a new instance of the object.


For example, we can define a three dimensional point as such:

function Point(x, y, z){
     this.X = x;
     this.Y = Y;
     this.Z = z;
}

This function can now be used like this:

var point = new Point(50,10,30);
alert(point.X); //50
point.X = 0;
alert(point.X); //0

Functionality can also be added by using the prototype object to create a method.

This is done like so:

Point.prototype.dst  = function (p2){
     return Math.sqrt( Math.pow( this.X - p2.X ) + Math.pow( this.Y - p2.Y ) + Math.pow( this.Z - p2.Z )  );
}

The prototype can now be used like this:

var p1 = new Point(10, 15, 20);
var p2 = new Point(3, 1, 21);
var distance = p1.dst(p2);

Methods can also be defined internal within the object's constructor through the 'this' keyword: 

function Point(x, y, z){

     this.X = x;
     this.Y = y;
     this.Z = z;
     
     this.equals = function(p2){
          return ( this.X == p2.X && this.Y == p2.Y && this.Z == p2.Z );
     }
}

var p1 = new Point(10, 15, 20);
var p2 = new Point(3, 1, 21);

console.log(p1.equals(p2)); //false
