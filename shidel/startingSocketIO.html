Google Launcher Tutorial


Real time interaction is something that we have become accustomed to with mobile applications and many websites. With many server side languages this kind of functionality can be hard to achieve, and costly. however Node.js is built from the ground up for this sort of functionality, and with the help of a framework like socket.io it can be almost painless.


I am going to walk through some of the basics for using socket.io in this article. The first thing that you need to do is have node installed on your computer. there are many good guides on this topic, like this one.


Next you will need to install socket.io using node package manager. The command to do this will be npm socket.io.
